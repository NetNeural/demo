[2.2.0] — 2026-03-03
================================================================================
Epic #383 — Customer Data Export API (complete)

NEW FEATURES
  - Customer Export API (#384–387)
    • GET /functions/v1/export/telemetry   — paginated telemetry with cursor
    • GET /functions/v1/export/devices     — device inventory with filters
    • GET /functions/v1/export/devices/:id — single device + status history
    • GET /functions/v1/export/alerts      — alert data with severity filters
    • Bearer token auth (nn_live_ API keys), org-scoped queries
    • CSV export via Accept: text/csv on all routes
    • Keyset pagination with base64 cursor for all collection endpoints

  - Rate Limiting (#389)
    • Sliding 60-second window enforced per API key via api_usage_log table
    • 429 response when limit exceeded with Retry-After info
    • X-RateLimit-Limit / X-RateLimit-Remaining / X-RateLimit-Reset /
      X-RateLimit-Window response headers on every export request
    • Usage auto-pruned after 7 days

  - Webhook Subscriptions (#388)
    • POST/GET/PATCH/DELETE /functions/v1/webhook-subscriptions
    • HMAC-SHA256 payload signing (X-NetNeural-Signature header)
    • Supports events: alert.created, alert.resolved, device.online/offline/warning
    • HTTPS-only endpoint enforcement
    • Test-ping action to validate endpoint connectivity
    • Delivery log with status code, duration, success/fail tracking
    • Auto-dispatch on alert.created via send-alert-notifications function

  - OpenAPI Documentation (#390)
    • GET /functions/v1/export/openapi.json — OpenAPI 3.0 machine-readable spec
    • GET /functions/v1/export/openapi — Swagger UI (browser, try-it-out)
    • No auth required for spec endpoints
    • Documents all routes, parameters, pagination, security schemes, CSV responses

  - Developer Settings UI (#391)
    • New "Developer" tab group in Settings (alongside Account/Security/Organization)
    • API Keys section — create, copy, revoke, rotate keys with scope selection
    • Webhook Subscriptions section — create/delete/test-ping, event-type checkboxes
    • Secret shown once at creation, copy-to-clipboard button
    • Link to live Swagger UI from settings page

NEW DATABASE TABLES (migration 20260303230000)
  - api_usage_log           — per-request rate-limit tracking, 7-day TTL
  - webhook_subscriptions   — org-owned webhook endpoints with signing secret
  - webhook_delivery_log    — delivery attempt history, 30-day TTL

Epic #325 — Project Hydra (closed)
  All 13 stories confirmed complete (#326–338)
