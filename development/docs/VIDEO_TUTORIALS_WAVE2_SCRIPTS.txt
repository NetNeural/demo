M03 - Device List and Actions (3-5 min)
Start route: /dashboard/hardware-provisioning
=================================================================
"In this tutorial, we’ll cover daily device list operations. Open Hardware Provisioning. Use the search bar to find a device by name or serial. Apply status and organization filters to narrow the list. Open row actions and select View Details. Return to list and clear filters. The goal is fast lookup and clean handoffs between operators. You now know the core list workflow for operations."

M05 - Device Types (3-4 min)

=================================================================

You now understand why type consistency matters for scaling operations."

M08 - Analytics (3-5 min)

M10 - Reports: Alerts (3-5 min)

=================================================================

Open one row detail and explain actionable fields. Use export if available. You can now generate alert trend evidence for weekly reviews and incident retrospectives."
